# IEC 60870-5-104 (port 2404)

IEC 60870-5-104 is the power-SCADA transport for European
transmission / distribution networks and many non-EU utilities. It
runs over TCP/2404 and carries IEC 60870-5-101 ASDUs inside an APCI
envelope.

## Probe

- Send a U-format TESTFR act frame (Control byte 0x43 — "test the
  link is alive"). The frame is 6 bytes: start byte `0x68` + length
  `0x04` + control `0x43 0x00 0x00 0x00`.
- A TESTFR con response (Control byte 0x83) or any U-format reply
  confirms the target speaks IEC-104.

## Proxy policy (default build)

Three APCI frame types:

- **U-format** (Control[0] low two bits = `11`) — unnumbered
  supervisory: TESTFR / STARTDT / STOPDT activation + confirmation.
  **CategoryRead**, forward.
- **S-format** (low two bits = `01`) — supervisory acknowledgments.
  **CategoryRead**, forward.
- **I-format** (low two bits = `00` or `10`) — information transfer.
  Carry the ASDU with potential Control-family type IDs (45..64
  single / double / regulating / bitstring / setpoint / operation).
  **CategoryWrite**, refused.

Refusal = STOPDT_act U-frame (Control byte 0x13). Per the spec, a
master that receives STOPDT_act ceases I-frame transmission;
retries hit the same refusal.

## Writes (`-tags offensive`)

ASDU-level classification (distinguishing a read type-ID 100
General Interrogation from a write type-ID 45 Single Command)
lands with the offensive-build IEC-104 WriteGatedHandler in F6+.

## Scope

- Electric utility transmission SCADA (RTU gateways).
- High-voltage substations, renewable-generation plant controllers.
- Impact: direct switching of breakers / disconnectors.
