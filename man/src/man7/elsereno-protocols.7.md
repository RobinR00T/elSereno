# NAME

**elsereno-protocols** — overview of protocols supported or planned

# DESCRIPTION

ElSereno ships a pluggable set of protocol modules for ICS / OT and
legacy-network exposure auditing. For the current list of compiled-in
plugins, run:

```
elsereno plugins list
```

The catalogue at **.context/protocols/** tracks the planned plugins,
their default ports, and the phase in which they are scheduled to land.
See the main **README.md** for a protocol / Shodan-query table.
