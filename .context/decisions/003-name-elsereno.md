# ADR-003: Project name "ElSereno"

## Context
We need a memorable, non-generic name that evokes the problem domain:
auditing exposed systems that others have left unlocked.

## Decision
Name the project **ElSereno** — the Spanish night watchman who carried a
keyring capable of opening every portal in the neighbourhood. Binary name
`elsereno`.

## Consequences
### Positive
- Distinctive; unlikely to collide with established security tools.
- Evokes authorised, careful surveying of what is already accessible.

### Negative / trade-offs
- Non-English; may need a short tagline in documentation.

## Alternatives considered
- Generic names like `icsaudit` or `otscan` were rejected as forgettable.
