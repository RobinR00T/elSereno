# Snapshot — v1.54.0: Beckhoff TwinCAT ADS plugin

Single-chunk cycle. Closes the v1.32+ C carryover. Adds
the read-only fingerprint plugin for TwinCAT ADS over
TCP/48898 — Beckhoff's PC-based control suite running on
CXxxxx IPCs, IPCxxxx, embedded controllers, EtherCAT
couplers, BC bus terminals.

Plugin count: 28 → 29. Most-requested missing fingerprint
since v1.25 closed the legacy-ICS trio.

## Chunks

### Chunk 1 — wire + plugin + register (`a57777f`)

`internal/protocols/twincat/wire/wire.go` (new):

  - **AMS/TCP framing** (6-byte header: 2 reserved
    bytes + 4 LE length).
  - **AMS routing header** (32 bytes: target NetID +
    port, source NetID + port, command ID, state
    flags, data length, error code, invoke ID).
  - **AMSPortRouter = 10000** — well-known router
    service every TwinCAT runtime listens on
    internally.
  - **CmdReadDeviceInfo = 0x0001** — standardised
    across TC2 + TC3.
  - **BuildReadDeviceInfo(targetNetID)** constructs
    the 38-byte request frame.
  - **ParseDeviceInfo(buf)** validates AMS/TCP
    framing, checks response bit + cmd, parses
    payload as error(4) + version(major(1) + minor(1)
    + build(2)) + name(16, NUL-padded ASCII).
  - 4 sentinels: `ErrShortFrame`, `ErrBadAMSTCP`,
    `ErrLengthMismatch`, `ErrNotADSResponse`.

`internal/protocols/twincat/twincat.go` (new):

  - **Plugin** struct with DialTimeout, IOTimeout,
    TargetNetID. Default() = 5s/3s timeouts, zero
    NetID.
  - **TargetNetID** is operator-overridable.
    Zero-value (all-zero NetID) is the heuristic for
    initial probes when the operator doesn't yet
    know the device's real ID — most runtimes answer
    ReadDeviceInfo regardless.
  - **Probe** sends ReadDeviceInfo, parses response,
    builds finding note "TwinCAT <name> <major>.
    <minor>.<build>" (e.g. "TwinCAT TCatRouter
    3.1.4024", "TwinCAT TCatPlcCtrl 2.11.2308").
  - **Scoring**: protocol_risk 80, exposure 70,
    auth_state 85 (TC3 supports auth but many
    fielded systems open), capability 30/70 based
    on detection, impact_class 75 (factory-floor
    blast radius), cve_exposure 10 (CVE-2020-12525,
    CVE-2022-23166, CVE-2023-37452).
  - **ProxyHandler** is fail-closed; offensive ADS
    write path is future cycle.

`cmd/elsereno/plugins.go`:

  - import + Register call. Plugin enumeration via
    `elsereno plugins list` includes twincat
    (TCP/48898).

## Test coverage (8 cases)

  - **BuildReadDeviceInfo** — frame layout pinned
    (length, target NetID, cmd, request flag set,
    response flag clear).
  - **ParseDeviceInfo_Happy** — version 3.1.4024 +
    name "TC3 PLC1" round-trip.
  - **ParseDeviceInfo_NULPaddedName** — short name
    "TCatPlcCtrl" trims trailing NULs.
  - **ParseDeviceInfo_ShortFrame** — < FrameMinLen
    yields ErrShortFrame.
  - **ParseDeviceInfo_BadAMSTCP** — bad reserved
    prefix yields ErrBadAMSTCP.
  - **ParseDeviceInfo_NotResponse** — request flag
    set yields ErrNotADSResponse.
  - **ParseDeviceInfo_ADSError** — error code 0x07
    (Unknown Group) yields ErrNotADSResponse.
  - **ParseDeviceInfo_LengthOverflow** — length >
    MaxBodyLen yields ErrLengthMismatch.

All pass; lint clean.
