# Snapshot — v2.30.0: GE-SRTP service-0x21 family expansion

Single-chunk cycle. Closes the v2.10+ "GE-SRTP service-0x21
follow-up" carryover. v1.28 chunk-2 already implemented the
service-0x21 builder + ParseLongStatus extractor; v2.30
expands the model-family prefix table so newer GE Automation
product lines (Mark VIe controllers, VersaMax-M, CIMPLICITY,
Series-90 umbrella, PAC-IO) are recognised.

## New prefixes

Added 7:

  - `VersaMax-M`  — VersaMax Micro sub-family (was matched
    only as `IC200` previously).
  - `CIMPLICITY`  — GE HMI line, occasionally on SRTP.
  - `IS220`       — Mark VIe Distributed Controller.
  - `IS215`       — Mark VIe legacy controller.
  - `MarkVIe`     — turbine controller (firmware string
    lacks the space the marketing name has).
  - `Series-One`  — Series One legacy.
  - `Series-90`   — Series 90 family umbrella.
  - `PAC-IO`      — PAC-IO modular controller.

Total prefixes: 7 → 15.
