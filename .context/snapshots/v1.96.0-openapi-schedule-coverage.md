# Snapshot — v1.96.0: OpenAPI coverage for schedule endpoints

Single-chunk cycle. The OpenAPI spec lagged behind: v1.70
(CRUD), v1.74 (Update), v1.77 (preview), v1.84/v1.86
(audit), v1.88 (delete audit), v1.89 (retention overrides),
v1.92 (runs), v1.93 (clone), v1.95 (bulk pause/resume) all
landed without spec entries. Downstream SDK generators
were silently missing 13 endpoints.
