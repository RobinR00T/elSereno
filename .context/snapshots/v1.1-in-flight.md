## Design patterns established (must stay consistent)

### Per-plugin `WriteGatedHandler` (chunk 1)

Every offensive proxy follows:
```go
type AllowedX struct { /* proto-specific */ }
func AllowlistHash(target string, a []AllowedX) [32]byte
func SessionMutation(target string, a []AllowedX) confirm.Mutation
type WriteGatedHandler struct {
    Target         string
    Allowed        []AllowedX
    Deriver        confirm.KeyDeriver
    Auditor        confirm.Auditor
    SessionConfirm confirm.Confirm
    authorised     bool
}
func (h *WriteGatedHandler) Authorise(ctx) error
func (h *WriteGatedHandler) Handle(ctx, client, upstream) error
```
- Allowlist hash MUST be order-insensitive (sort before hashing).
- Per-frame decisions consult `Allowed`; no per-frame Authorize.
- Refusal frames are protocol-native (IllegalFunction for
  modbus, AckData err-class 0x85 for S7, encapsulation status
  0x0001 for ENIP, etc.).
- modbus/s7/enip have full `Handle` loops landed; bacnet/dnp3/
  iec104/hartip/atg/fox have session primitives only — full
  `Handle` is a v1.2 carry-over.

### File-backed audit writer (chunk 2)

- Path: `~/.elsereno/audit.jsonl` (mode 0600, parent dir 0700).
- One JSON-encoded `audit.Entry` per line.
- Chain invariant: each entry's `prev_hash` equals the prior
  entry's `entry_hash`. Genesis has `prev_hash = GenesisPrevHash`
  (32 zeros).
- `audit.FileWriter.Append` is serialised by a struct mutex;
  resumes across process restarts by scanning the last line.
- `audit.VerifyFile(path)` walks the chain and returns
  `ErrChainBroken` on the first mismatch.
- `confirm.NewAuditor(writer, actor)` maps
  `confirm.AuditEvent` → `audit.Entry` with
  disposition/category/protocol/operation/target/payload_hash/
  reason embedded in the JSONB payload. Maps confirm.Category
  onto the existing `audit_log` event_type enum (no migration
  needed).
- DB-backed Writer is a v1.2 ticket.

### Offensive runtime (chunk 3)

- `cmd/elsereno/offensive_runtime.go`:
  - `newOffensiveRuntime(cmd, passphraseFile)` opens the vault +
    audit writer + wraps into a confirm.Auditor.
  - `rt.Close()` releases the file handle.
  - `currentActor()` → `os/user.Current().Username` → `$USER` →
    "operator".
  - `defaultAuditPath()` → `~/.elsereno/audit.jsonl` with the
    parent dir auto-created mode 0700.
- CLI pattern for a new offensive verb:
  ```go
  rt, err := newOffensiveRuntime(cmd, ppFile)
  if err != nil { return err }
  defer rt.Close()
  c := confirm.Confirm{AcceptsWrites, ConfirmTarget, ConfirmToken}
  err = theOp(ctx, …, rt.Vault, rt.Auditor, …)
  ```
- Every Authorize decision lands one line in the JSONL before
  the network I/O runs; failed writes short-circuit.

### Build-tag fence

Nothing under `offensive/**` compiles in the default build.
`cmd/elsereno/*_offensive.go` and `cmd/elsereno/*_stub.go`
split the CLI registrations with `//go:build offensive` vs
`//go:build !offensive`. The stub returns an empty set so the
default binary does not expose any of the offensive verbs.
