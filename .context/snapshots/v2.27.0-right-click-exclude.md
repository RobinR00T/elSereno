# Snapshot — v2.27.0: right-click tag-chip exclude

Single-chunk cycle. Adds a third tag-filter gesture
complementing v2.6 click + v2.19 Shift+Click.

## UX

  - **Click** → replace filter set with this tag (v2.6).
  - **Shift+Click** → toggle membership (v2.19).
  - **Right-click** → toggle membership AND switch op to
    `not_in` (v2.27 new).

The op selector in the tag-cloud reflects the chosen op
so it's visible after a right-click.

`title` attribute on each chip documents the three
gestures so the discoverability isn't hidden.

## Impl

  - `setScheduleTagExclude(tag, ev)` calls
    `setScheduleTagFilter(tag, {shiftKey: true}, "not_in")`
    + preventDefault on the right-click event.
  - `setScheduleTagFilter` gained an optional `forceOp` arg
    that updates `scheduleTagFilterOp` + the visible
    `<select>`.

## Build sizes

Unchanged across all 3 variants.

Adds ~25 LoC in dashboard.go.

## Carryover

  - **vNext**: keyboard shortcut (Alt+Click) for the same
    effect, for trackpad operators.
